# GameHost

`GameHost` is a Unity UPM package that provides a pure C# host core.

## Install (Git UPM)
Add this to `Packages/manifest.json`:

```json
{
  "dependencies": {
    "com.noname.gamehost": "https://github.com/bossbobofather-cpu/GameHost.git#v0.1.0"
  }
}
```
